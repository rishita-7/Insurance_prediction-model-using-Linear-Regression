# Insurance_prediction-model-using-Linear-Regression
Predicting insurance charges using Linear Regression model.

# Model training
The project implements a Linear regression model to predict the final insurance charges based on several input features.

# Model Evaluation
The performance of this model is evaluated with the help of r2_score and adjusted r2_score.

# Performance
 r2_score= 0.80
 adjusted r2_score=0.79
